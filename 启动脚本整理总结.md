## 📋 保留的脚本

### 有用的启动脚本（共 6 个）

#### 1. 同时启动前后端
- `start_dev.bat` - CMD 版本
- `start_dev.ps1` - PowerShell 版本

**功能：** 一键启动前后端，最方便

#### 2. 单独启动后端
- `启动后端.bat` - CMD 版本
- `启动后端.ps1` - PowerShell 版本

**功能：** 
- 检查目录是否正确
- 自动激活虚拟环境
- 使用 `python -m uvicorn` 避免模块路径问题

#### 3. 单独启动前端
- `启动前端.bat` - CMD 版本
- `启动前端.ps1` - PowerShell 版本

**功能：** 启动 Vite 开发服务器

## 🗑️ 删除的脚本

## 🎯 脚本用途

| 脚本 | 用途 | 推荐场景 |
|------|------|---------|
| `start_dev.bat/ps1` | 同时启动前后端 | 日常开发 |
| `启动后端.bat/ps1` | 单独启动后端 | 后端开发/调试 |
| `启动前端.bat/ps1` | 单独启动前端 | 前端开发/调试 |

## 💡 使用建议

### 日常开发（推荐）

```bash
# 一键启动
start_dev.bat
```

### 分别启动

```bash
# 终端 1
启动后端.bat

# 终端 2
启动前端.bat
```
